BomberPhone

Version 1.0

A classic-style Bomberman game built for Cloud Phone — running on standard Web APIs, no special SDK required.

Built with

  • HTML5 Canvas for rendering
  • Plain CSS & JavaScript — no frameworks
  • Keyboard-first design for feature-phone navigation

Design

Controls follow the T9 dpad convention (2/8/4/6 to move, 5 to fire, 0 to pause) with LSK/RSK softkey navigation, matching Cloud Phone's keyboard-driven UI.

Screen

Designed and tested for the smallest Cloud Phone resolution: 240×320.
